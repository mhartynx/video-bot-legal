Video Bot

Video Bot is a desktop creator tool that helps the authorized creator generate short vertical videos and publish them to TikTok and YouTube.

TikTok OAuth Callback
Video Bot runs locally on the creator's computer. The desktop tool opens TikTok OAuth in the browser, then uses the registered callback page to complete authorization.

How It Works

The creator runs Video Bot from their local desktop environment. The tool can generate a script, create voice audio, compose subtitles and visuals, export an MP4 video, and upload or publish that video through platform APIs after the creator grants authorization.

Desktop Login Flow

The desktop tool starts the TikTok Login Kit flow by opening the TikTok authorization URL in the creator's browser. After the creator approves the requested permissions, TikTok redirects to the registered OAuth callback page. The creator copies the authorization code back into the desktop tool, which exchanges it for an access token and stores it locally.

TikTok Integration

Video Bot uses TikTok Login Kit for OAuth authorization and TikTok Content Posting API to upload or publish videos to the creator's own TikTok account. For Direct Post, the tool queries creator information, initializes a video publish request, uploads the MP4 file, and checks publish status until completion.
