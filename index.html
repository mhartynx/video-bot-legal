Video Bot

Video Bot is a local creator tool that helps the authorized creator generate short vertical videos and publish them to TikTok and YouTube.

How It Works

The tool runs in the creator's local environment. It can generate a script, create voice audio, compose subtitles and visuals, export an MP4 video, and upload or publish that video through platform APIs after the creator grants authorization.

TikTok Integration

Video Bot uses TikTok Login Kit for OAuth authorization and TikTok Content Posting API to upload videos to the creator's account. Direct Post is used only after the creator authorizes the required permissions.
